Fake News Classifier

Welcome to the Fake News Classifier

Discover the truth behind news articles with our Fake News Classifier. This web application utilizes a machine learning model, specifically the Multinomial Naive Bayes algorithm, implemented in Python using the Scikit-learn library. The news articles are processed using Pandas DataFrames to facilitate efficient data manipulation.

Disclaimer: This tool is designed for educational purposes only and should not be used as a sole determinant of the authenticity of news articles. While the algorithm is trained on a diverse dataset, it may not capture all nuances, and the results should be interpreted with caution.

Input Page:

  • Enter a news article in the text area below, and our system will leverage the power of Natural Language Processing (NLP) and machine learning to classify it as either fake or true. The backend Python code employs the Multinomial Naive Bayes algorithm, a popular choice for text classification tasks.

  • Disclaimer: The classification provided is based on patterns identified in the training data. It may not be foolproof, and the accuracy is subject to the quality and diversity of the training dataset. This tool should not be used as a substitute for critical thinking and fact-checking.

Output Page:

The news has been classified! The backend Python code, backed by Scikit-learn, processes the input through a Multinomial Naive Bayes model trained on a dataset of labeled fake and true news articles. The result is a classification that provides insights into the reliability of the news content.

Disclaimer: The classification is based on statistical analysis and may not be infallible. It is recommended to verify news from multiple sources and exercise critical thinking. This tool is not intended to replace human judgment or journalistic standards.

As a developer, I've crafted a versatile web app combining data preparation, machine learning, and a user-friendly Flask interface. Here's the breakdown:

Data Prep: Employed pandas to merge CSV files (Fake.csv and True.csv) into a DataFrame, assigning labels (0 for fake, 1 for true). Leveraged NLTK for text preprocessing, including tokenization, lowercase conversion, and stopwords removal.

ML Model: Split dataset, using TfidfVectorizer for text-to-number conversion, and trained a Multinomial Naive Bayes model on transformed data.

Flask Web App: Established routes for main, input, about, and contact pages. Implemented classification for user input (/classify).

Work Section: Added potential for project showcase with a new route (/work) and template (work.html).

Web Server: Ran Flask on a simple HTTP server, dynamically obtained NLTK data directory, and set up listening on port 8080.

A versatile app uniting data science and web development, providing a seamless news classification interface and room for project showcase.
